# AlphaForge AI

An institutional-grade AI equity research platform.

## Features (Planned)

- Live financial statement ingestion
- Automated ratio analysis
- Discounted Cash Flow (DCF) valuation
- SEC filing analysis
- Multi-agent investment committee
- AI-generated equity research reports
- Excel financial models
- Interactive dashboard

## Tech Stack

- Python
- FastAPI
- PostgreSQL
- SQLAlchemy
- React
- OpenAI API
- Docker
